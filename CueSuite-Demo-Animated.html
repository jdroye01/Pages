Detailed Setup

A clean first-run path for
staff, volunteers, and admins.

This page is not just marketing. It is meant to explain the full onboarding sequence so a new team can get ChurchScore Live running without guessing which step comes first.

Step 1
Configure Live Sync
Open Settings, enter the sync server URL, and verify the status light changes from disconnected to connected.
Step 2
Create or Import Classes
Define your ministry group structure first so every new member has a clean destination from day one.
Step 3
Add Point Actions
Set up the repeatable scoring rules such as attendance, memory verse, guest invite, or leadership participation.
Step 4
Define Tiers
Create milestone names like Bronze, Silver, and Gold so progress can be celebrated clearly from the member screen.
Step 5
Load Members
Import a roster or add members manually, then verify they appear under the correct class in the Members tab.
Step 6
Run a Test Service
Award points, open a member detail view, confirm the tier logic, and generate a report before using it live.
